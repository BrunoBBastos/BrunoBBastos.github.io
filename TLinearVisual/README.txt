Programa pra visualizar uma transformação linear num plano cartesiano.

Atualização - 08/01/2021
1 - O aplicativo permite que o usuário escolha os elementos da matriz de transformação
		através dos elementos de entrada no campo "Matriz da Transformação".

2 - O botão "Enviar Matriz" transmite os elementos da matriz do usuário para o
		programa, isso pode ser verificado no canto superior esquerdo do quadro.
	Para visualizar a transformação, basta clicar no botão "Aplicar Transformação".

3 - As caixas de seleção "Autovetores" e "Escala" permitem visualizar ou não os
		elementos referidos no quadro. Os autovetores serão referentes à matriz
		mostrada no quadro.

4 - A posição do mouse, relativa ao quadro e sua resolução, está sendo exibida abaixo
		do quadro no formato '(x, y)'. Os valores são arredondados para números
		inteiros¹, mas isso pode ser ignorado pressionando o botão 'Control'².
	A resolução do quadro pode ser aumentada ou diminuída através do uso do scroll
		do mouse sobre o quadro.
	É possível adicionar pontos ao quadro através de um clique com o botão direito do
		mouse sobre uma posição onde ainda não hajam outros pontos*. Ao selecionar mais
		de um ponto, será criada uma forma a partir da ligação destes.

* - Note que a posição do mouse é arredondada para números inteiros como
	mencionado em '¹', uma posição mais específica pode ser escolhida pressionando
	'Control' como descrito em '²'.
